ZapTask – Task Management Application

ZapTask is a simple and efficient task management application designed to help users organize their daily activities. The app focuses on clarity and ease of use, allowing users to create and delete tasks without unnecessary complexity. Each task is linked with a scheduled time, and when the time is reached, the app prompts the user with a reminder message asking, “Have you done this task?” ZapTask also supports both dark and light themes, enabling users to switch between modes based on their preference for a comfortable experience.

Key Features

  • Create & Delete Tasks Users can easily add new tasks and remove completed or unwanted tasks.
  • No Task Editing Tasks are designed to be simple—once created, they can only be completed or deleted.
  • Timed Task Reminders Displays a reminder message like “Have you done this task?” when the scheduled time is reached.
  • Dark & Light Theme Support Allows users to switch between dark and light modes for better usability.
  • Clean & Minimal UI Focused on distraction-free task management with a simple interface.

Used Technologies

  • Flutter
  • Dart
  • Hive Local Storage
