Hoor Al Bahr is a modern menu showcasing website developed for a seafood restaurant to display their food items, categories, and pricing in a clean and visually appealing format. The website is designed to be lightweight, fast, and responsive, making it ideal for in-store displays as well as customer browsing on mobile devices.

The project is currently live and in use at www.hooralbahr.com .

Used Technologies

  • Python (Flask)
  • HTML5
  • CSS3
  • JavaScript

Key Features

  • Clean and modern menu layout
  • Category-wise food listing
  • Optimized for TV and tablet displays
  • Mobile-friendly responsive design
  • Fast loading and easy navigation
