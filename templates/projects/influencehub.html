InfluenceHub is an influencer marketing application that connects brands and influencers on a single platform. Users can sign up either as a brand or an influencer, enabling seamless collaboration for marketing campaigns. The application is built using Flutter and Dart, focusing on smooth user experience, real-time communication, and scalable architecture.

Key Features

  • User roles: Influencer & Brand Allows users to register and operate based on their role.
  • Real-time chat system Enables instant communication between brands and influencers.
  • Campaign collaboration workflow Simplifies campaign discussions and partnership coordination.
  • Modern and intuitive UI Designed for smooth navigation and user-friendly experience.
  • Cross-platform mobile support Runs seamlessly on both Android and iOS using Flutter.

Used Technologies

  • Flutter
  • Dart
  • Firebase
